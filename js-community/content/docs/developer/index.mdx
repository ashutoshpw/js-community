# Developer Guide

This documentation site is generated from MDX files in `content/docs` using Fumadocs.

## Documentation workflow

1. Add or update MDX files inside `content/docs`.
2. Run `npm run docs:dev` while authoring docs pages.
3. Run `npm run docs:build` to verify production generation.

## Validation commands

```bash
npm run lint
npm run build
npm test
```
