# Developer Guide

This section is for engineers working on the platform itself — setting it up locally, understanding the architecture, writing tests, and deploying to production.

## In this section

| Page | What it covers |

## Tech stack at a glance

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript 5 |
| Runtime | Bun (preferred) or Node.js |
| Database | PostgreSQL via Drizzle ORM |
| Auth | better-auth |
| Styling | Tailwind CSS v4 |
| Linting/Formatting | Biome 2 |
| Testing | Vitest 4 + React Testing Library 16 |
| Background jobs | Inngest |
| File storage | Vercel Blob |
| Email | Resend (via Inngest) |
| Docs | Fumadocs (MDX) |
