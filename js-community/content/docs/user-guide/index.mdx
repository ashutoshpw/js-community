## Getting the most out of the platform

- **Stay in the right category** — topics posted in the correct category get better engagement
- **Use tags** — cross-category tags like `react` or `typescript` help people find your topics
- **Search before posting** — the full-text search often surfaces an existing answer
- **Like generously** — likes help contributors earn higher trust levels and keep the Top feed useful
- **Bookmark actively** — save reference topics to find them later without relying on search
