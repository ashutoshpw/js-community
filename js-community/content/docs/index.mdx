# JS Community Documentation

Welcome to the documentation hub for the JS Community platform. These guides cover product onboarding, community moderation, developer workflows, and the APIs that support the platform.

## Explore the docs

- [Getting Started](/docs/getting-started) — install dependencies, launch the app, and learn the project structure.
- [User Guide](/docs/user-guide) — navigate the forum experience and common member workflows.
- [Admin Guide](/docs/admin-guide) — manage moderation tasks, categories, and platform settings.
- [API Reference](/docs/api-reference) — review the route handlers that support registration and forum actions.
- [Developer Guide](/docs/developer) — understand the docs workflow and validation commands used by contributors.
