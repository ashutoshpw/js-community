# Getting Started

JS Community is a modern discussion forum for JavaScript developers, built on Next.js 16, PostgreSQL, TypeScript, and Tailwind CSS v4. This section walks you through everything from creating an account to writing your first post.

## What you can do

- Browse and search thousands of topics across categories and tags
- Create threaded discussions with full Markdown support
- Reply, quote, and like posts from community members
- Send private messages directly to other users
- Receive real-time notifications when someone mentions or replies to you
- Upload images up to 5 MB (JPEG, PNG, GIF, WebP) to include in posts
- Earn higher trust levels as you participate, unlocking more features
