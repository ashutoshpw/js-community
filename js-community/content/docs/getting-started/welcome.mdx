# Welcome to JS Community

JS Community is a discussion platform for JavaScript developers. Whether you're learning the basics, exploring a new framework, or contributing to open-source — this is where you ask questions, share knowledge, and connect with other developers.

## What kind of platform is this?

JS Community is a **threaded forum**, similar in model to Discourse. Discussions are organized as topics inside categories. Each topic has an opening post followed by replies. You can:

- **Ask questions** — get help from experienced developers
- **Share knowledge** — post tutorials, links, and code snippets
- **Start discussions** — open-ended conversations about JS ecosystem topics
- **Give feedback** — comment on projects and ideas

## How content is organized

| Level | Description |
|-------|-------------|
| **Category** | A broad topic area (e.g., "JavaScript", "Tools", "Showcase"). Maintained by admins. |
| **Tag** | A label applied to a topic for cross-category grouping (e.g., `react`, `typescript`). |
| **Topic** | A thread started by a user. Has a title, category, optional tags, and an opening post. |
| **Post** | A message within a topic. The first post is the topic body; subsequent posts are replies. |

## Forum feeds

Three main feeds are available in the sidebar:

- **Latest** (`/forum/latest`) — topics ordered by most recent activity
- **Top** (`/forum/top`) — topics ordered by like count
- **Categories** (`/forum/categories`) — browse topics filtered by category

## What you cannot do as a new user

When you first register you are at **Trust Level 0 (New User)**. A few limits apply:

- Maximum 10 outgoing private messages per day
- Some rate limits on consecutive likes

These limits relax automatically as you read topics and participate. See [Trust Levels](/docs/user-guide/trust-levels) for full details.

## Community standards

All interactions are governed by the [Community Guidelines](/guidelines). The short version: be respectful, stay on-topic, and help others learn.
