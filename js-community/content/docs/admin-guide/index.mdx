## Accessing the admin area

Navigate to `/admin`. If your account does not have `admin: true` set in the `users` table, you will receive a 401 Unauthorized response.

To grant admin access to a user, update the database directly or use the user management panel if another admin account already exists:

```sql
UPDATE users SET admin = true WHERE username = 'your-username';
```

## Admin dashboard

The admin dashboard at `/admin` shows a summary of:

- Total users, topics, and posts
- New users, topics, and posts today vs yesterday (with trend %)
- Quick links to all admin sub-sections
